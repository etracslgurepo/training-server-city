==========================
A. Restore Databases
==========================

01. Open a Navicat or SQL Management Studio 
   and connect to your database engine

02. Open a Query window 

03. Load and execute the file "_res/_initial/mssql/00-create-eor-db.sql" 

04. Load and execute the file "_res/_initial/mssql/01-create-image-db.sql" 

05. Load and execute the file "_res/_initial/mssql/02-create-notification-db.sql" 

06. Restore the backup file "_res/_initial/mssql/training_etracs_city.bak" 
    using SQL Management Studio only

07. Load and execute the file "_res/_initial/mssql/04-insert-etracs-data.sql" 

08. Load and execute the file "_res/_initial/mssql/patches/01-patch-etracs-2.5.05.01-007.sql" 

09. Load and execute the file "_res/_initial/mssql/patches/02-patch-etracs-2.5.05.01-008.sql" 

10. DONE !!!



==========================
B. Modify DataSource Files
==========================

01. Go to folder "services/datasources"

02. Edit the file "epaymentds" and set the file content to: 

dialect=mssql
driverClass=com.microsoft.sqlserver.jdbc.SQLServerDriver
url=jdbc:sqlserver://${dbserver_host};DatabaseName=${db_epayment}
user=${dbserver_user}
pwd=${dbserver_pass}


03. Save and close the text editor.
